# Software Construction Assessment 1 | Priority Queue Class Library

The aim of the project is to implement and test different versions of the Priority Queue abstract data type in C#.

The following are included:

- SortedArrayPriorityQueue
- UnsortedArrayPriorityQueue
- UnsortedLinkedPriorityQueue
- SortedLinkedPriorityQueue
- HeapPriorityQueue

Projects in the solution :

- PriorityQueue
Contains all priority queue implementations.

- TestProgram
Handles a simple user interface which allows the user to test the implemented queues.

- NUnityPriorityQueueTesting
Tests for all queue implementations.

# How to run the program (from Visual Studio)

1. Set 'TestProgram' as the startup project.
2. Run the project.
3. Choose a queue implementation from dropdown.
4. Add items with priorities and use the buttons.
